Your folder structure

Now that you understand how computers organise files, it’s time to set up a neat folder structure for all your uni-related work.

1. Create a folder called "uni" in your Documents folder or your OneDrive folder (if you’re using OneDrive).

Here is where you will keep all files related to your study programme.

It’s up to you how you organise most of your folders and files within the "uni" folder but we strongly encourage you to give it some thought and keep your file and folder structure tidy.

2. Inside the "uni" folder, create a "psych_methods" folder.

This folder will contain all files related to your stats and methods modules. Notice the underscore (_) in the name. Get used to not using blank spaces in folder and file names. Computers like it like that!

Over the next two years, you will take four stats and methods modules. Let’s create a folder for each one.

3. Create four folders inside "uni/psych_methods":

  • 01_paas
  • 02_analysing_data
  • 03_disc_stats
  • 03_qqm

These folders are for the following modules (respectively):

  • Psychology as a Science (term 1)
  • Analysing Data (term 2)
  • Discovering Statistics (term 3)
  • Quantitative and Qualitative Methods (term 4)

Make sure you include the numbers with leading zeroes, followed by underscores. This way, your folders will be neatly sorted in the order, in which you will take the modules.

In this term, we will only be working with the "01_paas" folder but make sure you set up the structure now.

4. Finally, if your computer hides file extensions, make sure you disable this setting.

Because we will be working with quite a few different kinds of files, it is really important to know what the extension of any given one is. How to set your OS to show file extensions is shown in the video you just watched.

You can adopt the “termNumber_module_name” naming convention even for your non-methodsy modules, if you’d like. Alternatively, you can create one folder for each year of study ("y01", "y02", "y03") – each one hosting folders for individual modules – and one separate folder for your dissertation. But like we said, we will leave that to you.

Just please, whatever structure you come up with, make sure it’s logical and that you stick to it and keep your files organised. It will make your life a whole lot easier down the line!
