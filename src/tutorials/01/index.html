Text editor boxes

From week 3 on, we will start learning how to write documents in RStudio using its text editor, called R Markdown. Unlike visual text editor that you might be used to, such as Google Docs or Microsoft Word, R Markdown doesn‚Äôt have any buttons or icons you can press to make a part of the text bold or italicise it. Instead, it uses plain text characters such as *, _, #, ^, and others to format the final document.

To help you get used to this way of writing documents, the tutorials will sometimes let you practice R Markdown in text editor boxes such as the one below. Anything you type in the box will get rendered underneath as it would if you generated a document from a R Markdown file.
